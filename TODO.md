# FluxEngine Features

## Flow System
- Program flows with a simple Python-like language
- Rust support in later releases

## Node System
- Worker nodes: execute multiple tasks at once
- Flow nodes: sequential execution (queue system)
- Event nodes: process API/Webhook events (e.g., Weather API → desktop notification)
- If/Else nodes: conditional branching

## Code Export
- Export nodes as executable Python or Rust code
- Create standalone programs like FluxEngine

## Deployment (Future Releases)
- Local PC execution
- Self-hosted containerized deployment
